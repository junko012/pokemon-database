# Contributing

Contributions are welcome.

## Accepted Contributions

* Data corrections
* Schema improvements
* Documentation improvements
* Importer fixes
* Additional validation tooling

## Before Submitting

Please verify:

* Database integrity
* Foreign key consistency
* Learnset consistency
* Evolution consistency

## Pull Requests

Pull requests should include:

* Description
* Reasoning
* Validation performed
* Affected tables

## Bug Reports

Please include:

* Database version
* Query used
* Expected result
* Actual result
