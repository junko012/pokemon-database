# Roadmap

## Version 1.x

### Completed

* Species
* Moves
* Abilities
* Items
* Learnsets
* Evolution Chains
* Evolution Conditions
* Type Effectiveness

### Planned

* Form metadata expansion
* Encounter data
* Wild encounter tables
* Trainer data
* Regional dex mappings
* Localization tables

## Version 2.x

### Potential Features

* Competitive battle metadata
* Breeding metadata
* Egg group expansion
* AI-oriented query views
* Search optimization indexes

## Long-Term Goal

Become a portable, language-independent Pokémon knowledge database usable by:

* Desktop applications
* Web services
* AI systems
* Save editors
* Research projects
